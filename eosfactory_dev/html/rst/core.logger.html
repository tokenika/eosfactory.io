core.logger¶

eosfactory.core.logger.DEBUG(msg=None, verbosity=None, translate=True)[source]¶

DEBUG message logger.

Print the message, translated is the translate flag is set. Store the processed message in a buffer. The stored message is returned if the function is called empty.

Parameters

  • msg (str) – The message to be printed. If not set, return the buffer.

  • verbosity ([core.logger.Verbosity]) – The message is printed and buffered if, and only if, its name is in the verbosity list. If not set, the value set with the function core.logger.verbosity() is assumed, or a default value is assumed.

Welcome to EOSFactory’s documentation!¶

EOSFactory is a Python-based EOS smart-contract development framework, created by Tokenika. With a single command-line interface you can create a private testnet and then compile, unit-test and deploy EOS smart-contracts. All of this using simple yet powerful Python3 syntax.

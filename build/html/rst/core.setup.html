core.setup¶

eosfactory.core.setup.set_nodeos_address(address, prefix=None)[source]¶

Set testnet properties.

Parameters:
  • address (str) – testnet url, for example http://faucet.cryptokylin.io.
  • prefix (str) – A prefix prepended to names of system files like the wallet file and password map file and account map file, in order to relate them to the given testnet.

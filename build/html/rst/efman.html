efman¶

Python front-end for EOSIO cleos.

efman.account_map(logger=None)¶

Return json account map

Attempt to open the account map file named setup.account_map, located in the wallet directory wallet_dir(), to return its json contents. If the file does not exist, return an empty json.

If the file is corrupted, offer editing the file with the nano linux editor. Return None if the the offer is rejected.

efman.clear_testnet_cache(verbosity=None)¶

Remove wallet files associated with the current testnet.

efman.stop(verbosity=None)¶

Stops all running EOSIO nodes.

Return: True if no running nodes and the local nodeos wallet directory is empty, otherwise False.

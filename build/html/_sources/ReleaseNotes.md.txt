# Release Notes - v3.4.0

## Compatibility

EOSFactory [v3.4.0](https://github.com/tokenika/eosfactory/releases/tag/v3.4.0) is compatible with EOS [v1.8.0](https://github.com/EOSIO/eos/releases/tag/v1.8.0) and EOSIO.CDT [v1.6.1](https://github.com/EOSIO/eosio.cdt/releases/tag/v1.6.1).

## New features in v3.4.0

* Better control of installation dependencies for compatibility with EOSIO.

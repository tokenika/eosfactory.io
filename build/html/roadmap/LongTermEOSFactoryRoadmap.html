Long-term EOSFactory Roadmap¶

Support for other IDEs¶

Right now we are aiming for integration with Visual Studio Code (VSC), but we also consider supporting other IDEs, e.g. Eclipse or CLion.

Ricardian Contracts¶

We are thinking about integrating Ricardian Contracts into our unit-testing. This is a very interesting (and probably not widely known at this stage) aspect of EOS smart-contracts. For more information please refer to EOSIO documentation.
